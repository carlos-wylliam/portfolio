Projetos

To do list

To-Do List em HTML, CSS e Javascript, otimizado com LocalStorage para armazenamento local de tarefas. A interface amigável permite adição dinâmica e remoção de tarefas, com destaque para a marcação de conclusão.


Persistência de dados é garantida pelo LocalStorage, proporcionando funcionalidade mesmo após o fechamento do navegador. Simples, responsivo e eficiente, é uma solução prática para organização de tarefas diárias.
